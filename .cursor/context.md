This repository contains ESP‑IDF firmware for the M5StickC Plus 1.1.

Purpose:
Turn a DJI Osmo Action camera into a motion‑triggered dashcam remote.

Key hardware:

- ESP32‑PICO
- MPU6886 IMU
- ST7789 display
- BLE communication
